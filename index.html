GBG Better Web - 18th to 20th October, 2024

Gothenburg, Sweden

Welcome to Better Web 2024

Join us in Gothenburg for the Better Web Conference. Over three exciting days top industry experts and developers will come together to share knowledge, inspire innovation, and explore the future of the web.

The conference features 10+ seminars across various topics, designed to elevate your front-end development skills. Whether you're looking to stay on top of new technologies, improve performance, or enhance user experience, there's something for everyone.

What to expect in the coming days

The Future of CSS

This seminar highlights new CSS features like Grid Level 2, subgrid, and container queries that enable more flexible and responsive layouts. Attendees will also explore advanced color functions and CSS custom properties, equipping developers with tools for building dynamic, adaptable designs.

Accessible Web: Beyond Compliance

This seminar covers strategies for enhancing web accessibility for all users, including those with disabilities. Participants will learn about inclusive design principles, such as using semantic HTML and ARIA roles, as well as the importance of keyboard navigation. By focusing on user needs and best practices, developers can create websites that not only meet accessibility standards but also improve the overall experience for everyone.
