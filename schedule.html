Three days of awesome seminars

find your dates here

Day one - The 18th of October

The future of CSS

18/10 - 10:00

Explore the latest advancements in CSS, including features like CSS Subgrid, Container Queries, and Cascade Layers. Learn how these innovations will change the way we style websites and enhance flexibility in design.

JavaScript in 2030: What to Expect

18/10 - 13:00

A forward-looking seminar on the future of JavaScript, covering upcoming ECMAScript features and how the language is evolving to handle new challenges like machine learning, real-time applications, and more complex web interactions.

WebAssembly: Revolutionizing Web Performance

18/10 - 15:00

Discover how WebAssembly is transforming web development by enabling near-native performance in the browser. We'll explore its potential for gaming, heavy data processing, and how it opens doors to new programming languages on the web.

Day two - The 19th of October

Accessible Web: Beyond Compliance

19/10 - 10:00

In this session, we'll explore how to go beyond the minimum accessibility standards to create truly inclusive websites. Learn the future of accessibility in web development, focusing on user-centered design, cognitive accessibility, and emerging assistive technologies..

Headless CMS and the Decoupled Future

19/10 - 13:00

Dive into the growing trend of headless CMS and how it decouples the backend from the frontend. Discover how this architecture offers more flexibility, scalability, and performance in delivering content-driven web experiences.

AI-Driven Web Development: The Role of Machine Learning in Frontend

19/10 - 15:00

This seminar will discuss how AI and machine learning are influencing web development, from enhancing UX personalization to automating coding tasks. Learn how to leverage AI tools to streamline frontend workflows and improve user experiences.

Day three - The 20th of October

The Rise of Edge Computing for Web Developers

12/10 - 10:00

Learn how edge computing is reshaping the way we deploy web applications by bringing data processing closer to the user. We'll explore its impact on performance, latency, and security, and how it complements traditional cloud architectures.

Next-Gen Web Security: Securing the Web of Tomorrow

12/10 - 13:00

As web technologies advance, so do the threats. This session will focus on emerging web security challenges and solutions, including encryption, user privacy, and protecting web applications against new forms of attacks.

Building for the Immersive Web: AR/VR and 3D Experiences

12/10 - 15:00

Discover how augmented reality (AR), virtual reality (VR), and 3D elements are shaping the future of web development. Learn about the tools and frameworks you'll need to create immersive and interactive experiences for the next generation of web users.

Progressive Web Apps 2.0: The Future of Cross-Platform Experiences

12/10 - 17:00

PWAs are evolving to offer more native-like capabilities. This session will cover the latest advancements in PWA technology, including updates in offline support, push notifications, and performance improvements, making them a key player in the future of web apps.
